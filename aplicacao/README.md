# Cinebase

Cinebase é uma aplicação que utiliza o arquivo `movies_united.json` para exibir filmes, informações detalhadas e avaliações de diferentes fontes: **IMDb**, **Rotten Tomatoes** e **Letterboxd**.

Este projeto foi desenvolvido com **Vite**, **React** e **TypeScript**.

---

## Como rodar o projeto

1. Instale as dependências:

```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```
